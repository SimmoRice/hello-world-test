# Hello World Test

A simple static website project for testing the AI Scrum Master workflow.

## Purpose

This repository is used to test the distributed AI Scrum Master architecture with simple, incremental feature additions.

## Project Structure

- Static HTML/CSS/JavaScript website
- No build process required
- Open `index.html` in a browser to view

## Testing Workflow

Each issue represents a simple feature addition:
1. Issue #1: Create static "Hello World" page
2. Future issues: Add buttons, calculator, dark mode, etc.

## Development

Simply open `index.html` in your browser. No dependencies or build steps required.
